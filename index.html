CloudXR sample client config generator

このツールは NVIDIA CloudXR SDK ドキュメント のクライアントコマンドラインオプションを参考に作成されています。

プラットフォーム選択

サーバー接続

サーバーIP (-s, -server)
接続するサーバーのIPアドレス

表示モード

ウィンドウモード (-w, -windowed) Windows/Linux
デスクトップモードで実行する

ログレベル

詳細ログ (-v, -log-verbose)
より詳細なログを有効にする
ログ無効化 (-q, -log-quiet)
ログを無効にする
プライバシーフィルタリング無効化 (-p, -log-privacy-disable)
ログのプライバシーフィルタリングを無効にする
最大ログサイズ (-lmk N, -log-max-kb N)
最大ログサイズをキロバイト単位で指定（-1〜1000）

トレース設定

ストリームイベントトレース (-t, -trace-stream-events)
ストリーミングイベントのトレースを有効にする
ローカルイベントトレース (-tle, -trace-local-events)
ローカルイベントのトレースを有効にする
QoS統計トレース (-tqs, -trace-qos-stats)
QoS統計のトレースを有効にする

ネットワーク設定

クライアントネットワークインターフェース (-nic S, -network-interface-client S)
使用するクライアントネットワークインターフェースを指定
ネットワークトポロジー (-nt S, -network-topology S)
使用するネットワークトポロジーを指定
L4S輻輳制御 (-l4s, -enable-l4s)
L4S輻輳制御を有効にする

オーディオ設定

オーディオ送信 (-sa, -enable-send-audio)
サーバーへのオーディオ送信を有効にする
オーディオ受信無効化 (-dra, -disable-receive-audio)
サーバーからのオーディオ受信を無効にする
オーディオダンプ (-da, -dump-audio)
送受信オーディオをダンプする

ビデオ設定

最大解像度係数 (-m N, -max-res-factor N)
最大ストリーム解像度係数（範囲：0.5-2.0）
ビデオストリーム数 (-ns N, -num-video-streams N)
ビデオストリーム数を指定(カスタムサーバー向け)
グラフィックスタイプ (-g S, -graphics-type S)
グラフィックスコンテキストタイプを選択
アルファチャンネル (-a, -enable-alpha) Windows/Linux
アルファチャンネルを有効にする
フォビエイテッドスケーリング (-f N, -foveation N)
フォビエイテッドスケーリングを有効にする（値：25-100）
画像ダンプ (-d, -dump-images)
ストリーミング画像をダンプする
クライアントビットストリームキャプチャ (-ccb, -capture-client-bitstream)
クライアント受信ビットストリームをキャプチャする
サーバービットストリームキャプチャ (-csb, -capture-server-bitstream)
サーバー送信ビットストリームをキャプチャする

その他の設定

ボタン再マッピング無効化 (-b, -no-button-remap)
ボタンの再マッピングを行わない
レイテンシーテスト (-l, -latency-test)
ローカルレイテンシーテストを実行する
IRデコーダー無効化 (-ird, -disable-ir-decoder) Android
Almalence IRデコーダーを無効にする
フォールバックデコーダー (-fbd, -fallback-decoder)
フォールバックデコーダーを使用する
サーバー情報埋め込み (-esi, -embed-server-info)
サーバー情報を埋め込む
クライアント情報埋め込み (-eci, -embed-client-info) Windows
クライアント情報を埋め込む
ユーザーデータ (-u S, -user-data S)
ユーザー文字列をサーバーに送信
